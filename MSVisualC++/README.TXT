Набор программ для работы с телефонным оборудованием.
Основная задача программ - получение номера звонящего и передача его дальше по протоколу TCP/IP.
Может работать с оборудованием ISDN имеющим драйвер CAPI и с оборудованием подключённым по RS232 порту.
Тестировалось на работе с сотовыми телефонами подключёнными по COM порту через Bluetooth. 

31.03.2012 Определение номеров по CallerID для модем Acorp Sprinter@56K Ext.
